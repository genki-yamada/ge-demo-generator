外部データソース

MCP サーバーを追加してデモを外部データで強化

一般的なビジネスシナリオ向けに事前設定された MCP サーバーのバンドルを選択し、デモを素早く起動できます。

仕組み ▼
1
解析: GitHub URL を入力。Gemini が言語と環境変数をスキャン。
2
統合: 「デモに追加」をクリック。スクリプトに MCP サイドカー設定を含む。
3
入力: Cloud Shell でスクリプトを実行。API キーの入力を求められます。
4
デプロイ: キーは Secret Manager に保存、MCP はエージェントコンテナにクローンされサイドカーとして実行。
注意事項と制約 ▼
  • Python および Node.js/TypeScript リポジトリのみ対応。
  • 環境変数はターミナルで安全に入力。
  • OAuth やインタラクティブ認証フレームワークは非対応。
  • 大型ネイティブバイナリ（Chromium、FFmpeg 等）はスキップ。
技術アーキテクチャ ▼

Universal MCP Bridge アーキテクチャを活用:

  • ランタイム抽象化: 汎用プロキシでビルドパック依存を解消。
  • トランスポート調整: HTTP/SSE を stdio/SSE に自動ブリッジ。
  • 自動デプロイ: エージェントコンテナにクローンしサイドカーとして実行。
  • 自動バインディング: supergateway 経由でエージェントに内部接続。
Agent ──(HTTP/SSE)──▶ [Universal Bridge] ──(stdio/SSE)──▶ Custom MCP

GE Demo Generator の仕組み

このウェブアプリは、リソースをプロジェクトに直接デプロイしません。代わりに、入力内容に基づいてパーソナライズされたセットアップシェルスクリプトを動的に合成します。Google Cloud Shell でこのスクリプトを実行すると、必要なリソース（BigQuery、Firestore、Cloud Run）がお客様の Google Cloud 環境内に自動的にプロビジョニングされ、即座にデモを開始できます。

合成されたエージェントはデュアルモデルアーキテクチャを採用: Flash-Lite コーディネーター（root_agent）が低レイテンシで通常のインタラクションを担当し、Pro サブエージェント（deep_analysis_agent）が複雑なマルチステップ推論タスクに自動的に委譲されます。モデルは --model-analysis-agent と --model-root-agent CLI フラグで設定可能です。

さらに、このアクセラレーターはデータセットとともにドメイン固有のサンプルプロンプト、PDF、Excelファイルを生成します。これにより、リッチでリアルなエンドツーエンドのプリセールスデモをシームレスに体験できます。

    インタラクティブ A2UI によるエージェントワークフロー
    
            
    自律的な実行とリッチなインタラクティブ UI（A2UI）をシームレスに統合したエージェントワークフローを即座に生成。エージェントが非構造化ドキュメントを処理し、エンタープライズデータベースに対してデータを自動修正し、リアルタイムのヒューマン・イン・ザ・ループ判断のための動的 UI コンポーネント（カード、フォーム、承認フロー）を表示できます。ゼロコードで即座に構築された完全なビジネスオペレーションを視覚的に訴求することで、販売サイクルを加速し「これが欲しい」という瞬間を生み出します。

    カスタム MCP サーバーのインポート
    
            
    GitHub URL を貼り付けるだけで、任意の Model Context Protocol (MCP) サーバーを自動解析してデモに統合できます。デモエージェントをライブの顧客固有データソースとツールに即座に接続し、顧客の具体的な課題にオンザフライで対応する超パーソナライズされたシナリオを構築できます。
